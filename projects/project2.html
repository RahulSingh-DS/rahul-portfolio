Problem Statement

Build a system that recommends movies similar to a user's selection without requiring any user login or history. The system should run in real-time and be accessible via a clean web interface — no Netflix account required!

How It Works

01

Feature Extraction

Movie metadata — genres, keywords, cast, and crew — are extracted and combined into a single "tags" string per movie.

02

Vectorization

Tags are transformed into numerical vectors using Bag of Words (CountVectorizer), creating a matrix of movie feature vectors.

03

Similarity Computation

Cosine Similarity is computed between all movie vectors. Movies with the highest similarity scores are selected as recommendations.

04

Streamlit Deployment

The entire app is wrapped in a Streamlit interface — users select a movie and get top 5 recommendations instantly.

Features

🎥 Top 5 Similar Movies ⚡ Real-time Recommendations 🧠 No User Data Required 🌐 Live Web App 📊 Vectorized Similarity 🎨 Interactive UI

Deployment

The app is fully deployed on Streamlit Cloud and accessible publicly. The similarity matrix and movie data are serialized using pickle for fast load times without recomputing on every request.
